JSON-to-Go

Translate JSON into a rough Go type definition

  • JSON
  • Golang
What's This All About?

This tool helps when you want to avoid interface{} and type assertions or writing definitions by hand. The translater makes some assumptions, so be sure to check the results (and then use go fmt.)

Read more about JSON and Go

Usage

1. Type in a name for the Go type to generate.
2. Paste in your JSON.
3. Click the Convert button.
4. Grab your Go code from the Golang tab.
5. Rinse and repeat.
